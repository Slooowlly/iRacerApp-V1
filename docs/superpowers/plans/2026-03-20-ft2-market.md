# FT-2 Market Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** implementar o mercado de transferências persistido entre temporadas, com renovações, contratações IA, propostas para o jogador e garantia de equipes completas.

**Architecture:** o mercado será organizado em módulos pequenos sob `src-tauri/src/market/`, com structs compartilhadas em `proposals.rs`, regras isoladas em módulos puros e uma orquestração única em `market/pipeline.rs`. O `evolution/pipeline.rs` passa a chamar o mercado depois das aposentadorias e antes do reset de stats.

**Tech Stack:** Rust, rusqlite, serde, rand, Tauri backend, testes unitários com `StdRng`.
